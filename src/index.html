Citation Style Maintenance Demo

The time is ripe for a fresh approach to legal citation styles and their maintenance. The U.S. jurisdiction, in particular, is notorious for the hypertrophic detail of its citation requirements. Automated citation systems have a 35-year history in the sciences; but legal writing is as cumbersome today as it was in 1985. This page is where that will begin to change.

To remain current, a legal style manual must set guidelines for an expanding array of resources. To be useful, an automated system must be tuned to track those requirements with some precision. If the two projects are combined, the maintenance burden for both can be reduced, while significantly improving user experience—and the tools for doing so already exist.

The sample text below is drawn from the public-domain Indigo Book. Click the linked citation to see how the item is recorded in the open-source Jurism reference manager. Refer to this memo for a description of how maintenance of the human- and machine-readable versions of the style can be unified.
